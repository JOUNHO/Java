# 기본 API 클래스

## Object 클래스
* 객체 비교(equals())
* 객체 해시코드(hashCode())
* 객체 문자 정보(toString())
* 객체 복제(clone())
* 객체 소멸자(finalize())

## Objects 클래스
* 객체 비교(compare(T a,T b,Comparator<T>c))
* 동등 비교(equals() 와 deepEquals())
* 해시 코드 생성(hash(), hashCode())
* 널 여부 조사(isNull(),nonNull(),requireNonNull())
* 객체 문자정보(toStirng())

## System 클래스
* 프로그램 종료(exit())
* 쓰레기 수집기 실행(gc())
* 현재 시각 읽기(currentTimeMillis(),nanoTime())
* 시스템 프로퍼티 읽기(getProperty())

## Class 클래스
* Class 객체 얻기(getClass(),forName())
* 리플렉션(getDeclaredConstrucotrs(),getDeclareFields(),getDeclaredMethods())

### String 메소드
* 문자추출(charAt())
* 문자열 비교(equals())
* 바이트 배열로 변환(getBytes())
* 문자열 찾기(indexOf())
* 문자열 길이(length())
* 문자열 대치(replace())
* 문자열 잘라내기(substring())
* 알파벳 소.대문자 변경(toLowerCase(),toUpperCase())
* 문자열 앞뒤 공백 잘라내기(trim())
* 문자열 변환(valueof())
